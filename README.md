CommandBlockerPro is a feature-rich command blocking plugin for PaperMC (and forks of PaperMC) servers running Minecraft 1.12.2 or later. It's designed for servers trying to take their configuration to the next level, with the ability to set multiple different filters based on different conditions, with configurable block messages that support [PlaceholderAPI](https://www.spigotmc.org/resources/placeholderapi.6245/).

## Features
* Ability to define multiple rules with different permissions.
* Block Full Commands.
* Block commands with a given string in them.
* Block commands based off regex.
* Whitelist specific commands based off the above conditions.
* Hide specific commands with the same conditions.
* An API to interact with the plugin through.
* [MiniMessage](https://github.com/KyoriPowered/adventure) support
* [PlaceholderAPI](https://www.spigotmc.org/resources/placeholderapi.6245/) support
* [BetterReload](https://www.spigotmc.org/resources/betterreload.111256/) support

## Requirements
* Java 21 or newer
* Server Software with Paper API.
* Minecraft 1.12 or later
* [PlaceholderAPI](https://www.spigotmc.org/resources/placeholderapi.6245/) (Optional)

## Commands
| Command | Aliases | Permission Node | Description |
| ------- | ------- | --------------- | ----------- |
| **/commandblocker** | **/cb**, **/cbp** | **commandblocker.admin** | **Main plugin command** |
| /cb help | **/cb h** | N/A | View a link to the plugin wiki |
| /cb version | **/cb v** | N/A | View the current version of the plugin |
| /cb reload | N/A | N/A | Reloads the configuration file |
